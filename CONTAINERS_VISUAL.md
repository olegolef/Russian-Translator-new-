## 🎨 Легенда цветов

- 🟦 **Синий** - Основные контейнеры (Container, Box)
- 🟨 **Желтый** - Навигационные контейнеры (Toolbar, Tabs)
- 🟩 **Зеленый** - Контентные контейнеры (Content, Typography)
- 🟪 **Фиолетовый** - Специализированные контейнеры (Paper, Card)
- 🟧 **Оранжевый** - Главные контейнеры (Main Container)

## 📋 Список всех контейнеров

### **Основные контейнеры:**
1. **AppBar Container** - Заголовок приложения
2. **Main Container** - Главный контейнер с maxWidth="lg"
3. **Content Box** - Контейнер основного контента
4. **Text Reader Box** - Контейнер для чтения текста
5. **Book Catalog Paper** - Контейнер каталога книг
6. **File Upload Paper** - Контейнер загрузки файлов

### **Навигационные контейнеры:**
7. **Toolbar Container** - Панель инструментов
8. **Tabs Container** - Контейнер вкладок
9. **TabPanel Container** - Контейнер содержимого вкладки
10. **Navigation Buttons Container** - Контейнер кнопок навигации

### **Контентные контейнеры:**
11. **Text Container** - Контейнер текста (изменяемый размер)
12. **Scrollable Content Box** - Скроллируемый контент
13. **Typography Container** - Контейнер текста
14. **Tooltip Container** - Контейнер тултипа перевода

### **Специализированные контейнеры:**
15. **Grid Container** - Сетка для книг
16. **Card Container** - Карточка книги
17. **Card Content Container** - Содержимое карточки
18. **Card Actions Container** - Действия карточки
19. **Upload Content Box** - Контент загрузки
20. **File Types Box** - Типы файлов

### **Боковые панели:**
21. **Side Panels Container** - Контейнер боковых панелей
22. **User Dictionary Panel** - Панель словаря
23. **Formatting Panel** - Панель форматирования
24. **Comments Panel** - Панель комментариев
25. **Navigation Panel** - Панель навигации
